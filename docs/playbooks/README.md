# Playbooks Comerciais — Renegocia

Guias operacionais para o comercial/SDR da Renegocia rodar a jornada do cliente de ponta a ponta. Escritos em linguagem simples — não precisa ser advogado tributarista pra executar.

## Para quem é

SDR / comercial contratado para trabalhar os leads que chegam pelo site ou por prospecção ativa. Você **não precisa** responder dúvida jurídica específica — você qualifica, agenda, apresenta, fecha e passa pro stakeholder técnico executar.

## Quando acionar o stakeholder técnico

Sempre que:
- O cliente faz pergunta jurídica específica ("mas nesse caso o STJ decidiu…")
- Precisa validar se uma tese se aplica ao caso concreto
- Vai enviar proposta (a assinatura é do stakeholder técnico)
- Reunião de diagnóstico aprofundado

Na prática: você qualifica → agenda diagnóstico → stakeholder técnico conduz o diagnóstico → você segue no pós (proposta, follow-up, fechamento, kickoff).

## Princípios da Renegocia no relacionamento comercial

1. **Honestidade numérica.** Se a estimativa de crédito é R$ 8k, diz R$ 8k. Não infla.
2. **Sem custo inicial, só êxito (nas teses de recuperação).** É a nossa maior vantagem — repita sempre.
3. **Diagnóstico gratuito é sério, não chamariz.** Entregamos valor real na primeira reunião, mesmo que o cliente não feche.
4. **Não prometer prazo de decisão judicial.** Estimar com base em precedentes, nunca "garantir".
5. **Todo contato termina com próxima ação combinada e data.** Nunca deixar em aberto.
