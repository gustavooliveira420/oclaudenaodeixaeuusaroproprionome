# Bibliografia de Cases e Fontes Externas

Toda referência externa usada em post do `@renegocia.tributario` é registrada aqui antes da publicação, conforme [CONTRATO-SEMANAL.md](CONTRATO-SEMANAL.md) seção 3.

Política curta: jurisprudência e notícia são públicas → adapta. Copy de competitor → não usa.

## Formato

```
### Post #{N} ({data publicação})
- **Fonte:** {link}
- **Tipo:** decisão STF | decisão STJ | notícia | release Receita | ...
- **Acessado em:** {YYYY-MM-DD}
- **Trecho relevante:** {1-2 linhas do que foi usado como ponto de partida}
- **Adaptação Renegocia:** {1 linha do que mudou na versão publicada}
```
